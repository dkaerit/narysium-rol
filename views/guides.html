Organizar clases


Para organizar clases cada profesor debe iniciar un nuevo twit con un formato como el siguiente: CLASE DE HERBOLOGÍA

Fecha - 25 de Septiembre
Hora - 17:30 (+00:00)
Hashtag - #ClassHerbNSR

Responde debajo si planeas asistir ese día. La idea es que cada miembro del fake que tenga intención de participar en la actividad elaborada por los profesores responda a dicho twit. La cuenta madre siempre citará el encabezado de esos hilos y los colgará del fijado para que todo el mundo pueda estar al tanto de clases que se hayan programado.

Cada profesor puede hacer una lista de twitter con los miembros que decidan participar para que puedan tener una tl propia dedicada, esto es opcional, pero se recomienda.

Lo que si deben hacer es, cuando comience la actividad, utilizar el hashtag para rolear dicha actividad, por ejemplo. #ClassHerbNSR

Andaba distraído sin prestar demasiada antención a las indicaciones del profesor mientras nos guiaba por el bosque.


Tratad de no hacer coincidir fechas para clases con las tramas globales del fake.



Tirar dados

Esto es una guía básica para el uso de los comando de los dados al mencionar a la cuenta de activades. Antes que nada, importantísimo: A veces el servidor que se encarga de lanzar dados se queda en reposo cuando pasa un tiempo que no es usa por eso, la primera vez que vayan a lanzar un dado en el día, antes de nada debéis de entrar en esta página:

https://narysium-act.herokuapp.com/

Dicha página también esta en el perfil de la cuenta de actividades. Ahora, una vez explicado todo esto, comencemos con el uso del comando.

Lo primero, para activar la llamada de un dado se debe mencionar a la cuenta de actividades (@narysiumact) con el comando !r

Se pueden hacer 3 tipos de tirada:

  • Tirada ventajosa
  • Tirada en desventaja
  • Tirada normal
